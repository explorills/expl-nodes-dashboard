# expl-nodes-dashboard
EXPL Nodes - Web3 Dashboard, Ethereum-based, staking, liquidity pool/mining, node management, etc. built with React, TypeScript, Material-UI, and wagmi

## Overview

This dashboard provides a user interface for interacting with EXPL nodes and related blockchain operations. Built for Ethereum-based workflows, it supports staking management, liquidity pool operations, and node monitoring.

## Technology Stack

- React 19 with TypeScript
- Material-UI v7
- Vite
- wagmi for Ethereum interactions

## Installation

Install dependencies:

```bash
yarn install
```

Create environment configuration:

```bash
cp .env.example .env
```

Edit `.env` with required configuration values.

## Development

Start development server:

```bash
yarn dev
```

Build for production:

```bash
yarn build
```
Built using [Mantis MUI React](https://codedthemes.gitbook.io/mantis) (v4.0.0).
